# 🌸 MusicBouquet v5 Photo PRO

Мини-сервис для создания красивых музыкальных подарков.

---

## ✨ Возможности

- 🎁 Создание персонального подарка
- 💌 Тёплое поздравление
- 📷 Фото получателя
- 💐 Выбор букета
- 🎵 Добавление музыки:
  - YouTube
  - Яндекс Видео
  - MP3 ссылка
- 🎨 Авто-выбор фона по тексту
- 📱 Полностью адаптивный дизайн
- 🚫 Без базы данных (localStorage)
